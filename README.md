# Laboratorio #1 - Landing Page Responsiva

## Descripción

Este proyecto corresponde al **Laboratorio #1** del curso **ISW-521 - Programación en Ambiente Web I**.

Consiste en el desarrollo de una **Landing Page Responsiva** para la presentación de un portafolio profesional, aplicando buenas prácticas de desarrollo web mediante HTML5, CSS3 y JavaScript.

## Objetivos

- Utilizar HTML5 semántico para estructurar el contenido.
- Implementar diseños responsivos mediante Flexbox y CSS Grid.
- Aplicar principios básicos de accesibilidad web.
- Implementar persistencia de datos utilizando Web Storage.
- Desarrollar una interfaz adaptable a dispositivos móviles, tabletas y computadoras.

## Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript
- Flexbox
- CSS Grid
- LocalStorage


## Funcionalidades

### Diseño Responsivo

La página se adapta a diferentes tamaños de pantalla mediante Media Queries.

### Accesibilidad

- Uso de etiquetas semánticas HTML5.
- Imágenes con atributos `alt` descriptivos.
- Navegación mediante teclado.
- Uso de atributos ARIA donde corresponde.
